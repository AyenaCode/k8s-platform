## Application Design and Build

This public CKAD domain is weighted at **20%**. It tests whether you can choose
the right Kubernetes object and shape a Pod around the application.

You will practice:

- choosing a workload resource instead of forcing everything into a Pod
- selecting exact container images and commands
- using init containers and sidecars with a shared volume
- using both `emptyDir` and `PersistentVolumeClaim` volumes

Use namespace **`ckad-design`** for every task in this module.
