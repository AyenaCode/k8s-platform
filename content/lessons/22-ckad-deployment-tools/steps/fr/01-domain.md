## Application Deployment

Ce domaine public CKAD pese **20%**. Il cible la facon de livrer ou modifier une
application une fois la forme du Pod connue.

Tu vas pratiquer :

- rolling updates sur Deployments
- bascule blue/green avec selectors
- canary release par ratio de replicas
- installation d'un chart Helm local existant
- application d'un overlay Kustomize

Utilise le namespace **`ckad-deploy`** pour chaque tache.
