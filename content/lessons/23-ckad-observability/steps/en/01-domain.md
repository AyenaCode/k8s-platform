## Application Observability and Maintenance

This public CKAD domain is weighted at **15%**. It is about proving what the
application is doing, then changing the right Kubernetes field.

You will practice:

- readiness and liveness probes
- `kubectl logs` as evidence
- `kubectl describe` and rollout debugging
- migrating removed API versions to the current form

Use namespace **`ckad-observe`**.
